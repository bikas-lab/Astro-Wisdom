# Astro Wisdom — Astro Coach Landing Page Designs

Landing page design concepts for the **Astro Coach Certification** program by **Puneet Nahata** — India's first program uniting Jyotish, modern psychology and professional coaching.

Each design ships with a polished **light & dark mode**, custom SVG graphics, real imagery and working interactions (countdown, accordion, FAQ, theme toggle).
